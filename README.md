# Customer Churn Prediction using Machine Learning

## Growfinix Machine Learning Internship

### Task 3: E-Commerce – Customer Churn Prediction

## Objective

The objective of this project is to predict whether a customer is likely to stop using an application or cancel a subscription based on their purchase history, app usage and engagement score.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

## Machine Learning Model

* Random Forest Classifier

## Project Workflow

1. Import the required libraries.
2. Create a sample customer dataset.
3. Check for missing values.
4. Select input features and target variable.
5. Split the dataset into training and testing sets.
6. Train the Random Forest model.
7. Predict customer churn.
8. Calculate and display the model accuracy.

## Output

The program displays:

* First five rows of the dataset
* Missing value information
* Random Forest model accuracy
* Task completion message
